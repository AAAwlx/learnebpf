## 🎤 录制技巧

### 语速和停顿

**正常语速**：150-180 字/分钟

**停顿时机**：
- 切换场景前：停 2-3 秒
- 展示输出后：停 3-5 秒
- 重要概念前：停 1-2 秒
- 总结回顾前：停 2-3 秒

### 终端操作

**打字速度**：中等，不要太快

**命令建议**：
- 使用简短命令
- 避免复杂参数
- 预先测试可用

**错误处理**：
- 自然地："哦，让我重新试试"
- 解释原因："这是因为..."
- 继续演示，不要纠结

### 视觉提示

**重要内容**：
- 稍微放慢语速
- 重复关键信息
- 用手势/光标指示

**过渡内容**：
- 快速带过
- 不要详细展开
- 及时进入下一主题

## 🆘 常见问题处理

### 问题：输出为空

**现象**：bpftrace 没有输出

**应对**：
1. 检查是否有网络流量
2. 使用演示助手生成流量
3. 查看日志文件分析
4. 说明："现在没有活跃流量..."

### 问题：权限错误

**现象**：Permission denied

**应对**：
1. 展示 sudo 权限检查
2. 使用 sudo 运行命令
3. 说明原因："需要 root 权限..."

### 问题：AI调用失败

**现象**：API错误

**应对**：
1. 立即切换到本地分析
2. 说明："AI 模式需要 API，我们用本地模式"
3. 强调两种模式都很有用

### 问题：时间不够/过长

**现象**：超时/时间不足

**应对**：
- 过长：删除演示助手的等待时间
- 不足：快速带过部分内容，或减少一个场景
